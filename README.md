# Design System & Component Library
## The Aura Design System, a comprehensive foundation

## Overview

This project presents a small Design System created in Figma to ensure consistency, scalability, and reusability across user interface designs. The system includes design tokens, reusable components, and usage guidelines for maintaining a cohesive visual language.

---

## Goal

Create a small design system consisting of:

* Color Tokens
* Typography System
* Buttons
* Form Elements
* Cards

The design system helps designers and developers build interfaces faster while maintaining visual consistency.

---

## Features

### 1. Color Tokens

Defined a set of reusable color variables for:

* Primary Color
* Secondary Color
* Background Color
* Surface Color
* Text Colors
* Border Colors

### 2. Typography

Typography scale includes:

* Display Text
* Headings
* Subheadings
* Body Text
* Captions

Font sizes, weights, and hierarchy are clearly defined.

### 3. Spacing Tokens

Standardized spacing values:

* 4px
* 8px
* 16px
* 24px
* 32px
* 48px
* 64px

These tokens ensure consistent layout and alignment.

### 4. Buttons

Reusable button components with variants:

* Primary Button
* Secondary Button
* Disabled Button

### 5. Form Elements

Common form components including:

* Text Input
* Dropdown
* Text Area
* Error State Inputs

### 6. Cards

Reusable card components for displaying content:

* Feature Card
* Information Card
* Testimonial Card

---

## Usage Guidelines

### Do's

* Use primary buttons for main actions.
* Follow the predefined typography scale.
* Maintain consistent spacing using spacing tokens.
* Use approved color tokens only.

### Don'ts

* Do not introduce random colors.
* Do not use inconsistent font sizes.
* Do not modify component dimensions without necessity.
* Do not mix multiple design styles within the same interface.
